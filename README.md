### 信息结构化与知识管理工具

#### 需求说明

我现在需要一种从各种结构化与非结构化数据中快速提起信息，与信息间关联的工具。

需求分为以下三个部分：

1. 自动化数据捕获系统

通过公网API，本地的文件库或者对象存储的监控，通过定时轮询或者实时事件获取各种来源的数据；

尤其是需要通过局域网获取同网络下windows主机中的文件，并比对文件哈希用于判断文件是否变更，方式主要通过smd或者ssh等合法方式；

2. 自定义数据处理管线

对每一种不同的非结构化或者结构化的数据类型定义对应的数据处理管线；

例如对于所有的图片类型，使用yolo或者rcnn等小模型判断图片的种类并打上标签，对于携带有文字的类型进行OCR，识别出文字；
对于街景等现实照片使用yolo对图片中的拥有的目标进行识别；对于截图等信息定制化识别内容，一点一点的提取出图片中蕴含的信息。

对于PDF这类文件，先分析其中的文字是图片类型还是原生的文字，如果是图就OCR，如果是文字就分词分段提取；对于文档中的图标也要有对应的处理方法；

对于音频数据，先分析类型，再提取文字，最后按照类别定制提取信息

对于办公三大件也要找到对应的分析方式，目前还没有想好；

对于二进制文件也要提取数据，目前详细的处理方式还没有想法；

但是整体的信息结构就是对所有数据打标签，建立一个复杂的标签系统，结构化数据与数据之间的关系；
